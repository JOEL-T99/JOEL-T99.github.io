Java（十一）日期操作

Created2021-01-25|Updated2021-01-26
|Post View:|Comments:

日期操作

Java中的时间

  • Java中的时间使用标准类库Date类表示，是用距离一个固定时间点的毫秒数，表达一个特定的时间点

Date类的介绍

  • java.util.Date类封装日期及时间
  • Date类的大多数用于进行时间分量计算方法已经由Calendar取代

setTime / getTime

  • void setTime(Long time)

    用于为一个Date对象设置其需要表示的时间

  • Long getTime()

    获取一个Date对象表示的时间点，1970.1.1 00:00:00距离此时的毫秒数

Date重写toString

  • Date重写同 String方法，用一个字符串来描述当前Date对象所表示的是时间

SimpleDateFormat

  • 是一个以与语言环境有关的方式来格式化解析日期的类

日期模式匹配字符串

字符 含义 实例
y 年 yyyy年
M 月 MM月
d 日 dd日
E 星期
a 上下午 am上午、pm下午
H 24小时制 HH点
h 12小时制 hh点
m 分 mm分
s 秒 ss秒

Date与String转化

  • 将Date格式转化为String

    String format(Date)

  • 将String格式转化为Date

    Date parse(String str)

 

Calendar

Calendar简介

  • java.util.Calendar类用于封装日历信息，其主要作用在于时间分量的计算
  • Calendar是抽象类。本身不能被实例化

getInstance方法

  • Calendar getInstance()

    该方法根据当前系统所在的区域获取一个适当的Calendar

设置日期及时间分量

  • void set(int fidel, int value)

    可通过对不同的时间分量设置不同的值

获取时间分量对应的值

  • int get(int field)

    可以通过对不同的时间分量获取相应的值

getActualMaximum()

  • 用于获取给定时间分量所允许的最大值

ADD

  • void add(int field, int ammount)
