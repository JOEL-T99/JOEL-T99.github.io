数组的扩容

数组的长度在创建后是不可更改的。所谓扩容实际上是指创建一个更大的新数组并将原有数组的内容复制到其中。

数组的排序

  • 冒泡排序

    比较相邻的元素，如果违反最后的顺序准则(升序或降序),则交换位置。可以简化的理解，第一次找到所有元素中最大(最小)的放在最后一个位置上，不再变动；第二次找到剩余元素中最大(最小)放到倒数第二位不再变动，以此类推。
